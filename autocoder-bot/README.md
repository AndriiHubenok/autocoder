### Weather Analyzer

This Python tool allows you to analyze local temperature data from a CSV file. Follow the steps below to use the analyzer:

1. Make sure you have Python 3.14 installed on your system.
2. Install the required libraries by running `pip install -r requirements.txt`.
3. Run the `weather_analyzer.py` script by passing the file path to your CSV data as an argument to the `analyze_data` function.
4. Check the console for the mean, median, and standard deviation of the 'Temperature' column in the CSV file.
5. Find the line plot of the temperature data saved as `trends.png`.
